# job_agent
A fully autonomous Python agent that: 1. Reads your resume 2. Fetches job listings 3. Scores each job for fit (Agent 1 — Matcher) 4. Generates a tailored resume per job (Agent 2 — Generator) 5. Critiques and improves each resume (Agent 3 — Evaluator) 6. Saves polished results to `/outputs/`
